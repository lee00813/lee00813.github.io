HTML tag notes

Label and Input

They are used as a caption for html form and input is used to create a space where user can input their words, see example as below.

Tell me your name

Address

It is used for contact information, address, phone number and email

For example,

112 Maple Cres, Ottawa

Unordered List

It is used when a list does not have an order. For example,

List of my favourite cities

  • Sydney
  • Hong Kong
  • Tokyo
  • Montreal
  • Taipei

Link

Adding link to an external website or another page of your website. For example,

Check out my favourite song on Youtube
